# Neural Style Transfer
This project demonstrates Neural Style Transfer using deep learning. It combines the content of one image with the artistic style of another image to generate a stylized output.

## Technologies Used
- Python
- PyTorch
- Torchvision
- Matplotlib
- Google Colab

## How It Works
1. Load a content image and a style image.
2. Use a pretrained VGG19 model to extract image features.
3. Calculate content loss and style loss.
4. Optimize the generated image to combine both content and style.

## Output
The final image preserves the structure of the content image while applying the artistic style of the style image.
